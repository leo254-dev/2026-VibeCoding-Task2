關於我

你好！我是 Aiden，一名專注於創造直覺與美感併進的前端工程師。目前活躍於技術教學、中大型專案開發與企業接案。我擅長使用 React 與 TypeScript 構建高效能的 Web 應用，將設計稿精確轉化為流暢的數位體驗。

在過去三年的職業生涯中，我磨練出獨特的做事風格：我極度重視與團隊的深度溝通，熱衷於技術分享與教學，因為我相信把複雜的概念講清楚，是解決問題的首要步驟。
